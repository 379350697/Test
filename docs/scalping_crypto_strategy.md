# 剥头皮策略（Crypto 合约确认型）

> 策略名：`剥头皮`（别名：`scalping`）

## 1. 目标
将「关键位 + CVD 偏向 + 主动成交推动 + OI 结构 + 回抽确认」沉淀为可复现策略框架。

## 2. A / B / C setup 分级
- **A级**：关键信号完整（关键位清晰、CVD 偏向明显、推动有效、OI 代理支持、回抽确认到位、止损紧且 >=2R）
  - 处理：可用当日最大允许风险。
- **B级**：核心信号存在但强度一般（例如 CVD/推动/OI 其中一项偏弱）
  - 处理：正常风险。
- **C级**：信号模糊或冲突（关键位/背离/推动/回抽不充分）
  - 处理：轻仓或不做。

## 3. 系统实现映射（当前版本）
策略通过 K 线 + 成交量代理实现如下打分：

1. 关键位接近（最近 lookback 高低点附近）
2. CVD 代理偏向（`signed_volume / gross_volume`）
3. 主动成交推动代理（当根成交量 zscore + 价格位移）
4. OI 代理（短周期成交量均值 / 长周期成交量均值）
5. 回抽/回踩确认（回到突破参考位后再次确认）
6. 当根价格行为方向一致（实体方向）

当多/空得分中更高一方 >= `min_score` 时发出信号。

## 4. 盘前/盘中执行卡（可复现）
- 盘前只选 1-2 个标的：BTC/ETH（备选 SOL）
- 只做指定时段：欧盘后段、美盘开盘、重大数据前后
- 先判环境：趋势 / 震荡 / 消息 / 无序（不清楚则缩手）
- 先画关键位：昨高昨低、亚欧盘高低、箱体边界、供需区、最近推动区
- 入场前 8 问：关键位/CVD/大单/推动/OI/回抽/止损/2R
- 不满足 ≥2 项，直接放弃

## 5. 风控模板
- 单笔风险：C级 0.25R~0.5R，B级 0.75R~1R，A级 1R~1.5R
- 日内停手：连亏 3 笔或日亏损 2R 触发
- 达到 2R 优先减仓，3R 视结构处理
- 不允许亏损后加倍，不允许情绪化追单
